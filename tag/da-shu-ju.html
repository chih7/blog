实时计算框架Storm本地模式搭建

Posted on Mon 02 February 2015 in Tech • Tagged with linux, Storm, 大数据, 实时计算

平台：Ubuntu14.04

安装依赖

通过ubuntu自带的软件包管理器安装java环境。 安装Java：

$ sudo apt-get install openjdk-7-jdk

检查是否安装完成:

$ java -version

检查python版本:

$ python -V

安装ZooKeeper

为了避免每次获取超级权限重复输入sudo 临时进入超级用户:

$ sudo -s

使用ctrl+d退出

定位目录:

# cd /usr/local/src

从阿里的源下载:

# wget http://mirrors.aliyun.com/apache/zookeeper/zookeeper-3.4.6/zookeeper-3.4.6.tar.gz
