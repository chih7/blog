解决在opensuse下virtualbox无法挂载主机USB设备的问题

Posted on Mon 02 March 2015 in Tech • Tagged with linux, virtualbox, openSUSE, USB

因为要用virtualbox中的Windows虚拟机设置G602的可编程键，所以需要把G602的USB无线接收器挂载到虚拟机上，结果发现virtualbox右下角USB设备那里尽管有logitech USB Receiver这个选项，但它是灰色的，无法选中。

因为已经确认当前用户已经加入vboxusers组、并已经安装Oracle Extensions扩展包，那么问题应该从其他方向寻找。

在Google了一段时间后，终于在openSUSE论坛找到了解决方法

ubuntu突然出现打开系统设置慢 sudo时出现无法解析主机的解决办法

Posted on Sun 06 April 2014 in Tech • Tagged with Hosts, ubuntu, linux

突然间发现我的ubuntu12.04打开系统设置变得非常慢，正不知如何解决的时候。发现sudo时出现“无法解析的主机 XXXX" 的提示。马上就联想到不久前改过hosts，原来是主机名无法解析了，这样问题就好解决了

$ sudo vim /etc/hosts

加入以下行

127.0.0.1       （你现在的主机名）
127.0.1.1       （你现在的主机名）

之后 打开系统设置就非常快了

virtualbox主机访问虚拟机

Posted on Sun 02 March 2014 in Tech • Tagged with Hosts, virtualbox, linux, 网络

在virtualbox中，如果选择默认的网卡设置，也就是网络地址转换(NAT)，是不能让宿主机访问到虚拟机的，如果想让两者相互连通，可以在保留默认的NAT网卡的基础上增加一张Host-Only的网卡。

需要注意的是，如果你的宿主机是Linux的话，安装virtualbox后是没有一张Host-Only的网卡的，你需要在virtualbox的设置里添加这张网卡。
